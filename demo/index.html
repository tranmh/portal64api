API Information

Base URL

http://localhost:8080

Available Formats

All endpoints support both JSON and CSV output formats. Use the format query parameter or set the Accept header.

Authentication

No authentication required. This is a public API for chess rating data.

Rate Limits

No rate limits currently implemented.

Pagination

Search endpoints support pagination with limit and offset parameters. Maximum limit is 100 items per page.

Error Handling

Errors return appropriate HTTP status codes with JSON error messages containing details about the failure.
