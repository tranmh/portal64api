Spieler suchen

Suchen Sie nach Spielern anhand des Namens mit erweiterten Filter- und Sortieroptionen.

Individuelle Spieler-Abfrage

Schlagen Sie detaillierte Informationen für einen bestimmten Spieler mit seiner Spieler-ID nach.

Bitte geben Sie eine gültige Spieler-ID ein (Format: CLUBID-PERSONID)

Spieler-ID Format

Spieler-IDs folgen dem Format CLUBID-PERSONID, wobei CLUBID die Vereinskennung und PERSONID eine Nummer ist. Beispiele: UNKNOWN-10799083, C0101-1014, D300H-1234

Spieler-Bewertungsverlauf

Betrachten Sie den vollständigen DWZ-Bewertungsverlauf für einen bestimmten Spieler.

Bitte geben Sie eine gültige Spieler-ID ein (Format: CLUBID-PERSONID)

Vereinsspieler

Erhalten Sie alle Spieler, die zu einem bestimmten Schachverein gehören.

Bitte geben Sie eine gültige Vereins-ID ein (Format: C####)

Vereins-ID Format

Vereins-IDs folgen dem Format C####, wobei # für Ziffern steht. Zum Beispiel: C0101
