Individuelle Vereins-Abfrage

Schlagen Sie detaillierte Informationen für einen bestimmten Verein mit seiner VKZ/Vereins-ID nach.

Bitte geben Sie eine gültige Vereins-ID ein (alphanumerisches Format)

Vereins-ID Format

Vereins-IDs können in verschiedenen Formaten vorliegen: C#### (traditionell deutsch), B000E oder andere alphanumerische Codes. Beispiele: C0101, B000E, F2201

VKZ steht für "Vereins-Kennziffer" (Vereins-Identifikationsnummer) im deutschen Schachverbandssystem.
